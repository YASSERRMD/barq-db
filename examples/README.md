# Barq DB Examples

This directory contains examples for using Barq DB with various languages and scenarios.

## 1. RAG Pipeline (Python)
[View Example](./rag-python)

A Retrieval-Augmented Generation pipeline demo using Python, demonstrating how to ingest text, embed it, and query it for context injection.

## 2. Image Search (TypeScript)
[View Example](./image-search-ts)

An Image Search demo using TypeScript/Node.js, demonstrating how to store image metadata and vectors (mocked CLIP) and perform similarity search.

## 3. Logs Analysis (Rust)
[View Example](./logs-analysis-rust)

A high-throughput log ingestion and analysis demo using Rust, showing how to insert structured logs and query them efficiently.

## 4. WASM Benchmark (Web)
[View Example](./wasm)

A web-based benchmark running Barq DB's core index logic in the browser via WebAssembly.
